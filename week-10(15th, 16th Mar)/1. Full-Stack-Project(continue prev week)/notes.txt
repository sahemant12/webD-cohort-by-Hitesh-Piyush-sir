1. npm init: initialized project
2. to check about version: semver.org

3. if use require(): for importing file then in package.json make "type": "commonjs"
   if use import then in package.json make "type": "module"
   //help us to import file. Latest import is use.

4. command: npm i -D nodemon: for installing nodemon devDependencies
   in package.json scripts: write a script "start": "nodemon index.js" for using nodemon
   then run command: "npm run start", it will run the start script that start our server.

5. Zod, Yup: validation library

6. Nodemailer: this library is use for sending mail.

7. mailtrap: best for sending testing mail.

8. Resend: best for sending real mail.

9. clerk: use for authentication system.

10. bcrypt: encrypt the password.

11. apiDog: alternative of postman
